# Kinetic Text Generator

مشروع ويب بسيط يحاكي مولد النصوص المتحركة (مثل Space Type Generator Flash).

## المميزات
- إدخال نص مخصص.
- اختيار نوع الحركة (وميض، موجة، تكبير/تصغير).
- واجهة بسيطة وسهلة الاستخدام.

## الاستخدام
1. افتح الملف `index.html` في متصفحك.
2. اكتب النص الذي تريده.
3. اختر نوع الحركة من القائمة.

## ملاحظات
- يمكنك تخصيص الأنماط أو إضافة تأثيرات جديدة عبر تعديل `style.css` و`script.js`.
- الخطوط والتأثيرات المستخدمة هنا افتراضية ويمكنك استبدالها بخطوط أو تأثيرات أخرى.
